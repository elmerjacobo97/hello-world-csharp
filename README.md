# Aprendizaje de C#

Este repositorio contiene ejemplos de código que ilustran las bases de la programación en C#. Cada archivo de código fuente se centra en un concepto específico y es parte de mi proceso de aprendizaje en el lenguaje C#.

## Estructura del Proyecto

El proyecto está estructurado con varios archivos `.cs`, cada uno dedicado a temas particulares de C#:

- `App.cs`: Contiene el método `Main` y la lógica para ejecutar los ejemplos.
- `Variables.cs`: Demuestra la declaración y uso de variables.
- `Operadores.cs`: Ejemplos de operadores aritméticos, lógicos y de comparación.
- `Condicionales.cs`: Contiene ejemplos de estructuras condicionales como `if`, `else`, y `switch`.
- `Conversiones.cs`: Ilustra cómo realizar conversiones de tipos en C#.
- `Metodos.cs`: Muestra la definición y uso de métodos, incluida la sobrecarga de métodos.
- `Concatenacion.cs`: Proporciona ejemplos de cómo concatenar cadenas.

## Cómo Usar

Para ejecutar los ejemplos, necesitarás:

1. [Instalar .NET SDK](https://dotnet.microsoft.com/download).
2. Clonar el repositorio localmente usando `git clone [url-del-repositorio]`.
3. Navegar a la carpeta del repositorio clonado.
4. Ejecutar `dotnet run` desde la línea de comandos.
